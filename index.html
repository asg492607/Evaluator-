📋 Bulk Faculty Registration via Excel

📌 Excel Format Guide:
Upload an Excel sheet with the following 7 columns (header row required):

Faculty Name Div Batch 1 Batch 2 Batch 3 Email Number
Dr. Smith A ✓ ✓ smith@college.edu 9876543210

Rules:
  • Email → login username  |  Number → login password (min 6 digits)
  • At least 2 of 3 batches must be ticked per faculty row
  • Tick marks accepted: ✓, TRUE, YES, 1, x
  • Accounts are auto-approved — faculty can login immediately
